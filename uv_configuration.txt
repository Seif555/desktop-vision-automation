Environment: Windows 10 / Windows 11
Python: 3.9 - 3.11
How to install:
    python -m venv venv
    venv\Scripts\activate
    pip install -r requirements.txt

How to run:
    venv\Scripts\activate
    python main.py

Notes:
- The program writes output files to the current user's Desktop/tjm-project directory.
- The program expects `notepad_icon.png` next to `main.py`.
- Do NOT include the venv folder in the repository; use requirements.txt to reproduce the environment.
